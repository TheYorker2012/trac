Proper fix for #483 - make theyorker.co.uk redirect to www.theyorker.co.uk (already live).

Export .htaccess used %{SERVER_NAME} which is www.theyorker.co.uk from both theyorker.co.uk and www.theyorker.co.uk, HTTP_HOST.
It now redirects to %{SERVER_NAME} instead of explicit address.

Minor commenting in dev .htaccess.
