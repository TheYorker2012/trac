various calendar changes and fixes (with accompanying page properties)

#378 calendar event clashes with self between midnight and 5, fixed by calulating clashes before splitting events in that time.
when no notifications, display a note to say so
added links on calendar index page to next few days, this week, this term
fix defect in js calendar relating to timezone change. it was adding the number of seconds in a day. now does so, then offsets by the difference of timezone offsets.
added some styles to events such as attend,noattend,personal
fixed owned instead of owner in calendar backend, calendar event, user status
improved date_uri uri to accept thisterm, thisweek, thisday
disabled adding of events in js when don't have permission, should work correctly in public calendar, student calendar, vip calendar, directory calendar, etc

i think thats it!
