Message:

Work on date URIs:
Date_uri library:

Created library for creating and reading dates encoded in URIs.

View_listings library:

Makes use of Date_uri library and has shortcut generation function.
Now generates prev and next uri automatically using Date_uri.

View_listings_days library:

Sends eventHandlerJS data to view (empty string) so that it doesn't put a php error message as javascript.
Stores the number of days, and extends Retrieve() by setting prev and next uri using ViewListings::GenerateUri?.

View_listings_select_week library:

Now sets week links using ViewListings::GenerateUri?.

Listings controller:

Tidied constructor (loading lots of libraries).
Week function now gets date using Date_uri instead of from parameters.
_ShowCalendar parameters altered to take URI base and format instead of Presentation array.
_GenerateWeekUri and _GenerateWeekPresentation removed as now longer necessary.

Directory controller:

Events function now gets date using Date_uri and sets the URI base and format of the week select and events list view.
