Directory makes use of NavigationBar? class (I'm aware this will probably change, but it isn't certain how yet).
yorkerdirectory controller:

Sets up nav bar using frame_directory library in a single function. So data relating to where you can get to from the directory is all in this controller instead of in the view as well.

frame_directory library:

Now uses a more general frame_navbar library.

frame_navbar library:

Frame with a navbar (pretty simple, but may be used in multiple places).

directory_frame view:

Calls navigation bar loader instead of raw html.
