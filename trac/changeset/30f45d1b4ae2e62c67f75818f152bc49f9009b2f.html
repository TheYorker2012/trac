#385 - inline page property edit mode

enable from admin/pages
disable from admin/pages or from the inline edit stuff of a page property

models/pages_model:

added mInlineEditAllowed with value used to be in mInlineEditMode. this is whether the user has permission to be in inline edit mode.
changed mInlineEditMode to be set when mInlineEditAllowed and theres a variable inline_edit in session.
added SetInlineEditMode? function for enabling/disabling inline edit mode
added link to disable inline edit mode in the extra html for wikitext page properties

views/admin/pages_index

on pages index page, added inline edit mode section to sidebar with enable/disable links depending on state

controllers/admin/pages

get extra page property for text about inline edit mode on index page
added admin/pages/inline/{on,off}/{redirect/path}. to enable/disable inline edit mode using the new function in pages_model (which checks permissions)
